  • Setting up a LAMP Stack (Linux, Apache, MySQL, PHP)

    December 23, 2016

    1. Install Apache 2. Install MySQL 3. Install PHP 4. Verify LAMP Stack 1. Install Apache Install the package: Debian/Ubuntu sudo apt-get install apache2 Fedora/CentOS/SUSE Systems yum install httpd And start the service: sudo service apache2 start or sudo service httpd start   2. Install MySQL Debian/Ubuntu $ sudo apt-get install mysql-server $ sudo mysql_secure_installation $ sudo mysql_install_db $ sudo service mysql start CentOS/Fedora/SUSE $ sudo yum install mysql-server $ sudo service mysqld start $ sudo /usr/bin/mysql_secure_installation The mysql_secure_installation command runs the setup for MySQL.
